    Adversarial Autoencoders;
    Research Project

    Worked with a partner to reproduce results from the paper “Adversarial Autoencoders” by Makhzani et al, which introduced a generative model for image synthesis. To implement the architecture, we created an adversarial autoencoder model using three basic model blocks: an encoder, decoder, and discriminator. The model was trained, tuned, and tested on the MNIST dataset, which consists of 60,000 images. Finally, we wrote a conference-style technical paper describing our research and experiments.

    Tags: Generative AI, Image Synthesis, Guassian Distribution, Deep Learning

    "Blob Bodies"

    In the entertainment industry, marker-based 3D tracking is a widely used method for motion capture in video games and animated films. While highly accurate, it requires expensive proprietary equipment, is time consuming to set up, and is overall less accessible to everyday artists. As an alternative, pose estimation has gained popularity as a low-cost and open source method for body tracking. Therefore, this project investigated the application of Style Transfer to generate body tracking art in conjunction with Google's Blazepose and pose estimation algorithms.

    Tags: Image Segmentation, Pose Estimation Algorithms, Style Transfer

    Kaggle Competition
    Movie Prediction System

    Collaborated in a 5-person team to create a movie score prediction system that ranked in the top 5 of all team submissions. The objective was to implement an ML model which outputs the expected movie rating given user history. We started by preprocessing the 45,000 lines of data, and performed feature engineering using Pandas. Overall, we achieved a RMSE score of 0.177 using Singular Value Decomposition (SVD), a matrix factorization technique, and Optuna, an automatic hyperparameter optimization framework.

    Noise Creates Noise

    In the Generative Algorithms for Art & Architecture Course, my team was challenged to create a generative work that results in a physical medium. We decided to use a simple robot - an AxiDraw Pen Plotter, paper, ink, some code, and noise. These drawings were generated by a Python script that controlled the pen plotter’s movement. As the computer picked up noise from the environment, mathematical noise was added to the SVG drawing. We experimented with playing classical and rock music, and even singing!

    Emotion Reactive Photobooth

    For this project, my team created an AI photobooth that reacts to the user inside based on their facial expressions. By using Facial Emotion Recognition (FER) algorithms, our script analyzed and interpreted human facial expressions to determine the emotional state of an individual and change the patterns and colors inside the photo booth. This project was featured at the Cooper Union Summit on Generative Art, Architecture, and Engineering.
